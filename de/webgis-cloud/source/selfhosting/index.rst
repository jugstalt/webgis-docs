WebGIS selbst Bereitstellen
===========================

WebGIS ist eine Open Source Software und kann auf der eigenen Infrastruktur installiert werden.
Die Software kann dabei lokal für einen Single User Betrieb ähnlich einer Desktop Anwendung getrieben oder über das Intranet/Internet einem größeren Anwenderkreis
zur Verfügung gestellt werden.
