Authentication 
==============

In diesem Abschnitt wird gezeigt, wie die die Authentifizierung von Anwendern für das *WebGIS Portal* 
konfiguriert werden kann.

Die Authentifizierung erfolgt über die *WebGIS Portal* Applikation. Der Einstellung erfolgen daher 
in der ``portal.config`` Datei.
