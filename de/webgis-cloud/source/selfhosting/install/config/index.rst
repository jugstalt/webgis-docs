Konfiguration (Postinstallation)
================================

Über Konfiguration können die Webgis Applikationen zusätzlich den eigenen Bedürfnissen angepasst werden.

Die Konfiguration der Web Applikationen erfolgt in ``*.config`` Dateien, die bei der jeweiligen Applikation im Ordner ``_config`` liegen.
