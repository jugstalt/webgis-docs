WebGIS Cloud Resourcen (Details)
================================

In diesem Abschnitt wird das Erstellen und Verwalten der einzelnen *Resource Typen* genauer beschrieben.
Welche *Resourcen* zu Verfügung stehen, hängt von der jeweiligen *Subscription* ab, da hierfür auch Kosten (Credits) anfallen.
Für das Erstellen mancher *Resource Typen* muss auch noch eine Freischaltung des Portalbetreibers erfolgen, da diese nur mit 
Schulung/Einführung administriert werden sollten.
