WebGIS Cloud Portal
===================

Das *WebGIS Cloud Portal* dient zum Erstellen und Verwalten von Resourcen, die zum bereitstellen von WebGIS Karten
benötigt werden. Um das *WebGIS Cloud Portal* verwenden zu können, muss im ersten Schritt ein *Account* angelegt werden.
Dazu muss eine gültige E-Mail Adresse und ein Name angegeben werden. Die Verwendung des Portal in grundsätzlich
kostenlos. Für das anlegen von bestimmten *Resourcen* ist jedoch einerseits eine Einrichtungsgebühr sowie 
laufende Kosten zu bezahlen.

Das Verwenden der WebGIS API zum Einbinden einer Karte in die eigene Homepage ist kotenlos. Beim erstellen eines Accounts wird eine *"Free" Subscription
angelegt, in der nur *Resourcen* angelegt werden können, die kostenlos sind und bleiben. Als Kartenmaterial kann dazu 
auf die freien Hintergrundkarten von https://basemap.at zugegriffen werden. Ebenfalls kann eine Adresssuche (für Österreich)
eingebunden werden. 

Für komplexere Anwendungen (bishin zur Pflege der eigenen Geodaten über *Feature Service Dienste*) wird über 
erweiterte *"Credits" Subscriptions* ermöglicht. Die Kosten berechnen sich hier aus der Komplexität der Anwendung (=Anzahl 
der benötigten *Resourcen*) und den entstehenden Zugriffen.
