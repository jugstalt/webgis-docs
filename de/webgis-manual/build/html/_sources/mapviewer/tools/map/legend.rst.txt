Legende und Themen
==================

Dieses Werkzeug öffnet einen Dialog in dem die Legende für die aktuell dargestellte Karte angezeigt wird.
Ebenso kann in diesem Dialog die einzelnen Themenebenden anzeigen und schalten bzw. die Transparenz 
der einzelnen Kartendienste ändern.

.. note::
   Die Funktionsweise diese Werkzeugs entspricht dem *Menü* Button im *Darstellung*-Frame und wurde schon
   Punkt *Darstellung* in diesem Benutzerhandbuch beschrieben.
